$0: The Selected Element

Ever needed to reference the element you just clicked in the Elements panel? Meet $0 - your new best friend!

What it does: $0 refers to the currently selected element in the Elements panel.

Try it yourself:

  1. Right click and inspect this element
  2. Type $0 in the Console and press Enter
  3. Find the element's backgroundColor in rgb ?

Pro Tip: You can also access previously selected elements with $1, $2, $3, and $4!

$(): The Document Selector

Tired of typing document.querySelector() over and over?

What it does: $() is a shorthand for document.querySelector().

Let's compare:

          
// The old way:
document.querySelector('#my-awesome-id').textContent = 'Hello!';

// The cool way:
$('#my-awesome-id').textContent = 'Hello!';

Important: This console shorthand is NOT jQuery! It returns the first matching element only, not a collection.

Your turn: Try selecting an element with an ID on the current page:

$('#fancy-element-id-here')

$$(): The Multiple Selector

Need to grab all matching elements at once? $$() has got your back!

What it does: $$() is shorthand for document.querySelectorAll() and returns an array of elements.

For example, to get all buttons on a page:

        
  // Get all buttons
  $$('button')

  // Do something with each button
  $$('button').forEach(btn => btn.style.backgroundColor = 'purple')

Challenge: Count all paragraph elements on this page:

$$('p').length

$_: The Previous Result

Ever ran a command and then wanted to use its result? Meet $_ - the last expression's value!

What it does: $_ stores the value of the last evaluated expression in the console.

Try this sequence in your console:

          
// Step 1: Run this
'DevTools Ninja'

// Step 2: Now run this to see the previous result
$_

// Step 3: Let's manipulate the previous result
$_.toUpperCase()

// Step 4: And again!
$_.split('')

Power move: Chain operations on complex data without having to store variables!

Putting It All Together

Now that you know these powerful shorthands, let's combine them!

Real-world example: Change the background color of all paragraphs, then modify the first one specifically.

        
// Select all paragraphs and give them a light gray background
$$('p').forEach(p => p.style.backgroundColor = '#f0f0f0')

// Get the first paragraph
$('p')

// It's now in $_, so let's make it stand out!
$_.style.color = 'red'
$_.textContent = 'I was modified using console shortcuts!'

Key Takeaways:

  • $0 - Currently selected element in Elements panel
  • $() - Shorthand for document.querySelector()
  • $$() - Shorthand for document.querySelectorAll()
  • $_ - Value of the last expression

These shortcuts will make you significantly faster during debugging sessions!
