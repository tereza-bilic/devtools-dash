How to use the Flexbox Editor:

  1. Right-click any of the blue boxes above and select Inspect
  2. In the Styles panel, find display: flex on the parent container
  3. Click the flexbox badge (waffle icon) next to it
  4. A visual editor appears showing flexbox properties you can click to adjust!

Try this: Use the flexbox editor to change flex-direction to column and watch the items stack vertically!

The editor lets you visually control:

  • flex-direction (row/column)
  • justify-content (main axis alignment)
  • align-items (cross axis alignment)
  • flex-wrap behavior

Remember: You're changing the actual CSS in real-time! Any changes you make in the editor are immediately reflected in the Styles panel.

How to use the Grid Editor:

  1. Right-click on one of the orange boxes above and select Inspect
  2. In the Elements panel, find element with badge grid on the parent container
  3. Click the grid badge next to the element
  4. A visual overlay appears on your layout showing grid lines and tracks
  5. In the layout panel, you can toggle options to visualize different aspects of your grid

Cool feature: The grid editor shows line numbers! This makes it super easy to place items precisely using grid-column and grid-row properties.

Try this: With the grid editor active, try changing grid-template-columns to 1fr 2fr 1fr and see how columns resize proportionally!
